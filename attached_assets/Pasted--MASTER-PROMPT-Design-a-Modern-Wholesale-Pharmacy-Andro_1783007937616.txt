# MASTER PROMPT – Design a Modern Wholesale Pharmacy Android App

You are an expert Mobile UI/UX Designer and Senior Flutter App Designer.

Design a modern Android mobile application for a **Wholesale Pharmacy / Medicine Distribution Platform** similar to TrueMeds, Netmeds, PharmEasy, and IndiaMART Wholesale, but focused on **B2B medicine wholesalers, pharmacies, clinics, hospitals, and distributors**.

## Goal

Create a clean, premium, trustworthy, fast, and professional Android app UI.

Do NOT create backend code.

Only design complete UI screens with reusable components and proper navigation.

Use Material Design 3 guidelines.

The UI should be production-ready.
